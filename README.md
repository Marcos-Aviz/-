# 📋 Sistema de Avaliação de Alunos

Sistema para avaliação de alunos com observações positivas e negativas.

## 🌐 Acesse Online

👉 **[Clique aqui para acessar o sistema](https://SEU-USUARIO.github.io/SEU-REPOSITORIO/)**

> Substitua `SEU-USUARIO` e `SEU-REPOSITORIO` pelo seu usuário e nome do repositório no GitHub.

## ✨ Funcionalidades

- 📋 Lista completa com **172 alunos** (Nº 1001-1172)
- 🔍 Busca por nome ou número
- 📊 Ordenação por classificação ou nome
- ✅ Adicionar observações positivas
- ⚠️ Adicionar observações negativas
- 📱 Interface otimizada para celular
- 💾 Dados salvos no navegador (localStorage)
- 🔥 Suporte opcional a Firebase (sincronização na nuvem)
- 📤 Exportar dados em JSON
- 📥 Importar dados de backup

## 🚀 Como usar

### Opção 1: GitHub Pages (mais fácil)
1. Faça fork deste repositório
2. Vá em Settings → Pages → Source: GitHub Actions
3. O site será publicado automaticamente!

### Opção 2: Rodar localmente
```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
cd SEU-REPOSITORIO
npm install
npm run dev
```

### Opção 3: Com Firebase (sincronização na nuvem)
1. Crie um projeto no [Firebase Console](https://console.firebase.google.com)
2. Ative o Firestore Database
3. Edite `src/firebase/config.ts` com suas credenciais
4. Deploy novamente
